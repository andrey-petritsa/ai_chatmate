Система ai_chatmate - софт ассистент для общения в чатах, использующий LLM (текстовую нейросеть).
Код написан с применением принципов SOLID. Этот софт был создан в качестве эксперимента и проверки возможностей текстовых нейросетей.
Главная абстракция системы - обьект Platform. Каждая реализация платформы - плагин для приложения. При необходимости можно легко добавить новую платформу, написав новую реализацию Platform (WhatsappPlatform, FacebookPlatform, ViberPlatform, ...)
Код протестирован юнит и системными тестами.
